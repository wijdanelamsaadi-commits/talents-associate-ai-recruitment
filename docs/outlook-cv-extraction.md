# Outlook CV Extraction Note

For the first MVP version, CVs received by email in Outlook can be extracted with a simple VBA macro that saves attachments from selected messages into a local folder.

After the attachments are saved locally, recruiters can import those files through the platform CV upload workflow. This keeps the recruitment flow operational without blocking the MVP on mailbox API setup.

Microsoft Graph integration can remain an optional later enhancement for automatically reading mailbox attachments, applying permissions, and importing CV files directly from Outlook.
