# Frontend

React + Vite + TypeScript frontend skeleton for Talents Associate AI Recruitment Platform.

## Local Setup

```bash
npm install
copy .env.example .env
npm run dev
```

The frontend runs at `http://localhost:5173`.

Set `VITE_API_BASE_URL=http://localhost:8001` in `.env` when port `8000` is occupied.

## Build

```bash
npm run build
```
